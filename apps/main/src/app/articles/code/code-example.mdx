# Code Example

## Example without name

```tsx
import React from 'react';

function MyComponent() {
  return <div>Hello, world!</div>;
}
```

## Example with name

```tsx name="my-component.tsx"
import React from 'react';

function MyComponent() {
  return <div>Hello, world!</div>;
}
```

  return <div>Hello, world!</div>;
}
